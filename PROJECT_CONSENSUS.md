# 模板共识

这不是原来的卡牌数据 Mod 项目，而是一个已经抽离后的通用 STS2 Mod 模板。

## 保留了什么

- C# Mod 初始化入口
- 基础配置加载
- Harmony Patch 装配方式
- Windows-first 的构建 / 部署 / 打包 / 发布脚本
- 安装器和便携包产物链路

## 明确移除了什么

- 卡牌统计 UI
- 数据文件和 fallback 链路
- 采集器 / 同步器
- 遥测服务
- GitHub Pages 数据发布
- 自动更新和远程刷新逻辑

## 后续开发原则

- 优先做本地、独立、无外部依赖的 Mod 功能
- 需要联网、遥测、远程配置时，先确认真的是用户需求，而不是旧项目惯性
- 修改脚本时不要只修一条链路，要同时考虑 `deploy / portable / installer / release`
- 修改 `mod_manifest.json` 后，确认构建产物、安装器命名和发布命名都仍然正确
